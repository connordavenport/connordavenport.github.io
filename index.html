Connor Davenport is a typeface designer and writer of code for font development, currently based in Brooklyn, NY. To see his work check out GitHub, Mastodon, Instagram, or Process Blog.
Please reach out if you have any questions.

All of the typefaces are drawn in RoboFont.
My fonts would not be possible without libraries such as:
FontTools, Vanilla, Mojo, MutatorMath, Defcon, AFDKO, FontParts, as well as many others.
