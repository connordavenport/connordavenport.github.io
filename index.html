Connor Davenport is a typeface designer & writer of code for font development, currently based in Brooklyn, NY. To see his work check out GitHub, Twitter, Instagram, or Process Blog.
Please reach out if you have any questions.

All of the typefaces [shown on this site] were drawn in RoboFont. The font above is an experimental Variable COLRv0 Font, built using FreeType-PY.
My fonts would not be possible without libraries such as:
FontTools, Vanilla, Mojo, MutatorMath, Defcon, AFDKO, FontParts, as well as many others.
